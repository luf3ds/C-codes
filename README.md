# 💻 Códigos Diversos em C#

Este repositório reúne diversos exemplos, exercícios e pequenos projetos desenvolvidos em **C#**, com o objetivo de praticar lógica de programação, estrutura de dados, orientação a objetos e recursos da linguagem.

## 🚀 Sobre o Projeto

O repositório foi criado para armazenar códigos variados feitos durante estudos, testes e práticas com a linguagem **C#**.

Aqui podem ser encontrados exemplos simples e intermediários, envolvendo desde conceitos básicos até aplicações mais estruturadas.

## 🧠 Conteúdos Abordados

- ✅ Variáveis e tipos de dados
- ✅ Condicionais
- ✅ Laços de repetição
- ✅ Vetores e listas
- ✅ Métodos e funções
- ✅ Classes e objetos
- ✅ Encapsulamento
- ✅ Herança
- ✅ Polimorfismo
- ✅ Tratamento de exceções
- ✅ Manipulação de arquivos
- ✅ Exercícios de lógica
- ✅ Pequenos sistemas em console

## 🛠️ Tecnologias Utilizadas

- C#
